## Overview

Kristina Goshko provides a centralized documentation space where you can organize, manage, and collaborate on all your project documentation. Create structured guides, embed code examples, and maintain version history effortlessly. This default space serves as your hub for technical docs, API references, and onboarding materials.

## Quick Start

Get up and running in minutes. Follow these steps to create your first documentation page.

<Steps>
  <Step title="Create a New Page" icon="plus">
    Navigate to your space dashboard and click `New Page`. Choose the `introduction.mdx` template.
  </Step>
  <Step title="Add Content" icon="edit-3">
    Use the MDX editor to write your overview. Add headings, lists, and components like `<Callout>` for emphasis.

````mdx
## Welcome

Start writing here.
````
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Preview your page, then publish. Share the public URL: `https://docs.example.com/introduction`.
  </Step>
</Steps>

<Callout kind="tip">
  Customize your brand color to `#3B82F6` in space settings for a professional look.
</Callout>

<Tabs>

    Type queries like "API authentication" to surface relevant pages instantly.
  </Tab>
  <Tab title="Breadcrumbs" icon="chevrons-right">
